## 2. 문제 정의 및 해결 방안

### ❌ 기존 문제점
- **안전 관리 부족**: 해변 사고 발생 시 즉시 대응 어려움
- **혼잡도 파악 불가**: 실시간 인원 수 및 밀집도 정보 부재
- **수동 모니터링**: 24시간 CCTV 감시 인력 부족
- **정보 접근성**: 해변 정보 및 안전 수칙 전달 어려움

### ✅ 해결 방안
- **AI 기반 자동 감지**: YOLOv8로 사람, 쓰러짐 등 실시간 탐지
- **실시간 모니터링**: WebSocket 기반 실시간 데이터 전송
- **스마트 알림 시스템**: 이상 상황 발생 시 즉시 알림
- **AI 챗봇**: 24시간 해변 정보 및 안전 문의 응답

## 3. 시스템 아키텍처

### 🏗️ 전체 시스템 구조
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Frontend      │    │    Backend      │    │   Database      │
│   (Vue.js 3)    │◄──►│ (Spring Boot)   │◄──►│   (MySQL 8.0)   │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                       │                       │
         │                       │                       │
         ▼                       ▼                       ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   WebSocket     │    │   YOLOv8 AI     │    │   File Storage  │
│  Real-time      │    │  Detection      │    │   (Video/Image) │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 🔄 데이터 흐름
1. **CCTV 영상** → YOLOv8 객체 탐지
2. **탐지 결과** → 데이터베이스 저장
3. **실시간 데이터** → WebSocket으로 프론트엔드 전송
4. **사용자 요청** → REST API로 백엔드 처리

## 4. 핵심 기능 소개

### 🔐 사용자 인증 시스템
- **JWT 기반 인증**: 보안성 높은 토큰 기반 로그인
- **역할 기반 권한 관리**: ADMIN, MANAGER, USER 구분
- **BCrypt 암호화**: 안전한 비밀번호 저장

### 🏖️ 해변 관리 시스템
- **해변 정보 CRUD**: 생성, 조회, 수정, 삭제
- **지역별 검색**: 행정구역별 해변 정보 제공
- **상태 관리**: 활성/비활성 해변 구분

### 📹 AI 기반 CCTV 모니터링
- **실시간 스트리밍**: RTSP/HLS 프로토콜 지원
- **YOLOv8 객체 탐지**: 사람, 쓰러짐 등 실시간 감지
- **혼잡도 분석**: 인원 수 집계 및 히트맵 생성

### 🤖 AI 챗봇 서비스
- **OpenAI API 연동**: 자연어 기반 대화
- **해변 정보 제공**: 위치, 특성, 안전 수칙 등
- **24시간 응답**: 언제든지 문의 가능

### 🚨 안전 모니터링
- **쓰러짐 감지**: AI 기반 사고 상황 자동 감지
- **실시간 알림**: 이상 상황 발생 시 즉시 통보
- **이벤트 기록**: 사고 상황 추적 및 분석

## 5. 기술 스택 및 구현

### 🛠️ Backend 기술 스택
| 기술 | 버전 | 용도 |
|------|------|------|
| **Spring Boot** | 3.4.8 | 메인 프레임워크 |
| **Java** | 21 | 개발 언어 |
| **Spring Security** | - | 인증 및 권한 관리 |
| **Spring Data JPA** | - | 데이터 접근 계층 |
| **MySQL** | 8.0 | 데이터베이스 |
| **WebSocket** | - | 실시간 통신 |

### 🎨 Frontend 기술 스택
| 기술 | 버전 | 용도 |
|------|------|------|
| **Vue.js** | 3.5.18 | 프론트엔드 프레임워크 |
| **Vue Router** | 4.5.1 | 라우팅 관리 |
| **Pinia** | 2.1.7 | 상태 관리 |
| **Bootstrap** | 5.3.2 | UI 컴포넌트 |
| **Vite** | 7.0.6 | 빌드 도구 |

### 🤖 AI/ML 기술 스택
| 기술 | 용도 | 특징 |
|------|------|------|
| **YOLOv8** | 객체 탐지 | 실시간 고성능 탐지 |

### 🗄️ 데이터베이스 설계
- **8개 테이블**: 사용자, 해변, CCTV, 탐지결과 등
- **정규화**: 3NF 준수로 데이터 무결성 보장
- **인덱싱**: 성능 최적화를 위한 적절한 인덱스 설계
